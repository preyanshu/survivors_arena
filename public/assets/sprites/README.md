# Sprites Directory

Place your sprite images here:

- `player.png` - Player character sprite (64x64 recommended)
- `enemy.png` - Enemy sprite (64x64 recommended)
- `projectile.png` - Projectile/bullet sprite (32x32 recommended)
- `background.png` - Background texture (optional)

If images are not found, the game will automatically generate fallback sprites using canvas.

Supported formats: PNG, JPG, WebP
